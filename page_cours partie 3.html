Cours #3 : Personnaliser Votre Environnement

Personnaliser Votre Bureau

Changer le Fond d'Écran

  1. Cliquez avec le bouton droit sur le bureau.
  2. Sélectionnez "Personnaliser" ou "Fond d'écran".
  3. Choisissez une image parmi les propositions ou importez-en une depuis votre ordinateur.
  4. Enregistrez ou validez pour appliquer le changement.

Modifier les Couleurs

  1. Accédez aux paramètres ou à la personnalisation dans le menu Démarrer.
  2. Sélectionnez "Couleurs".
  3. Choisissez parmi les options proposées.
  4. Enregistrez ou validez pour appliquer les modifications.

Mettre à Jour le Système

  1. Accédez aux paramètres dans le menu Démarrer.
  2. Sélectionnez "Mise à jour et sécurité".
  3. Cliquez sur "Vérifier les mises à jour".
  4. Installez les mises à jour disponibles.

Gérer l'Espace de Stockage

  1. Cliquez sur "Ce PC" ou "Poste de travail".
  2. Faites un clic droit sur le disque dur principal et choisissez "Propriétés".
  3. Utilisez l'option "Nettoyer le disque" pour supprimer les fichiers inutiles.
