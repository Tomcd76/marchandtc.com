Cours #4 : Essentiel du Vocabulaire

Le basique des définitions

  1. Ordinateur : Une machine qui traite l'information.
  2. Système d'exploitation : Un logiciel qui permet à l'ordinateur de fonctionner.
  3. Bureau : L'écran principal où vous voyez les icônes et les fichiers.
  4. Fichier : Une collection d'informations stockées sur l'ordinateur.
  5. Dossier : Un endroit pour organiser les fichiers.
  6. Souris : Un outil pour déplacer le curseur à l'écran.
  7. Clavier : Un outil pour taper des lettres et des chiffres.
  8. Internet : Un réseau mondial de connexions informatiques.
  9. Site web : Une page ou un groupe de pages sur Internet.
  10. Navigateur web : Un programme pour accéder à des sites web (ex : Chrome, Firefox).
  11. Adresse URL : L'adresse d'un site web (ex : "www.exemple.com").
  12. E-mail : Messages électroniques envoyés sur Internet.
  13. Mot de passe : Un code secret pour sécuriser vos comptes en ligne.
  14. Mise à jour : Des améliorations pour les logiciels ou l'ordinateur.
  15. Virus : Un programme nuisible pour l'ordinateur.
  16. Logiciel : Programme informatique, comme Word ou Excel.
  17. Fenêtre : Un rectangle sur l'écran montrant un programme ou un fichier.
  18. Bouton : Un petit carré que vous pouvez cliquer avec la souris.
  19. Copier : Dupliquer des fichiers ou du texte.
  20. Coller : Insérer des fichiers ou du texte.
  21. Imprimer : Faire sortir une copie sur papier.
  22. Télécharger : Prendre des fichiers depuis Internet.
  23. Fichier ZIP : Plusieurs fichiers compressés dans un seul.
  24. Fichier PDF : Format de fichier pour partager des documents.
  25. Onglet : Des onglets dans un navigateur pour ouvrir plusieurs sites.
  26. Recherche : Utiliser un moteur de recherche pour trouver des informations sur Internet (ex : Google).
  27. Wi-Fi : Une connexion sans fil à Internet.
  28. Pilote : Logiciel pour faire fonctionner du matériel (ex : imprimante).
  29. Stockage : Endroit où vous gardez des fichiers (ex : disque dur, cloud).
  30. Sauvegarder : Copier des fichiers pour les protéger.
